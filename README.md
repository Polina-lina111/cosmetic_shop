## Основні сутності

### Category
Клас, який описує категорію косметичних товарів.

Приклади категорій:
- догляд за шкірою;
- макіяж;
- парфуми.

### Product
Клас, який описує косметичний товар.

Товар містить:
- назву;
- бренд;
- ціну;
- кількість;
- категорію.

---

## Зв'язок між сутностями

Одна категорія може містити багато товарів.

Приклад зв'язку: Category -> Product

---

## Реалізовані методи

### show_info()
Виводить інформацію про товар.

### update_price()
Змінює ціну товару.

### sell_product()
Зменшує кількість товару після продажу.

## Templates та detail view

У проєкті реалізовано сторінку detail для товару косметичного магазину.

### Реалізовано:
- detail view з параметром pk;
- передача context у template;
- використання template filters;
- цикл for для пов’язаних товарів;
- template inheritance через base.html.

### Використані template filters

#### upper
Назва товару виводиться великими літерами.

#### truncatewords:5
Опис товару скорочується до 5 слів.

#### date:"d.m.Y"
Дата відображається у форматі день.місяць.рік.

### Пов’язані товари

На сторінці detail виводиться список пов’язаних товарів:
- Lipstick
- Serum
- Face Mask

Якщо список порожній — виводиться:
"Нічого немає"
